# Game Taken

1. Speler-skin keuze: toon bij de startknop drie kleuropties voor het spelerblok
2. Obstakel-waarschuwingslijn: teken een rode stippellijn van boven naar beneden op de x-positie van een snel inkomend obstakel
3. Geluidsfeedback power-up verlopen: speel een aflopend geluid als invincible of speedboost afloopt
4. Mini-kaart: toon rechtsboven een piepklein overzicht (20×10px) van alle obstakelposities
5. Scherm-shake bij botsing: laat het canvas kort trillen wanneer de speler een obstakel raakt
