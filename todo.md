# Game Taken

2. Voeg een "slow motion" power-up toe die alle obstakels 3 seconden halveert in snelheid
3. Toon top-3 scores van de huidige sessie op het game-over scherm
4. Laat de levens-hartjes pulsen (rood knipperen) als er nog maar 1 leven over is
