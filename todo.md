# Game Taken

1. Visuele feedback bij level-up: laat het scherm kort oplichten in een andere kleur bij elk nieuw level
2. Obstakel-schaduw: teken een kleine schaduw onder elk obstakel voor meer dieptegevoel
3. Power-up timer: toon een kleine aftelbalk onder de speler wanneer een power-up actief is
4. Stijgende moeilijkheidsgraad: voeg bij level 5+ een tweede obstakel tegelijk toe per spawn
5. Score-multiplier indicator: toon een kleine x2/x3 badge naast de score wanneer een combo actief is
