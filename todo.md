# Game Taken

1. Obstakel-waarschuwingslijn: teken een rode stippellijn van boven naar beneden op de x-positie van een snel inkomend obstakel
2. Geluidsfeedback power-up verlopen: speel een aflopend geluid als invincible of speedboost afloopt
3. Mini-kaart: toon rechtsboven een piepklein overzicht (20×10px) van alle obstakelposities
4. Scherm-shake bij botsing: laat het canvas kort trillen wanneer de speler een obstakel raakt
