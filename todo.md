# Game Taken

1. Obstakel-schaduw: teken een kleine schaduw onder elk obstakel voor meer dieptegevoel
2. Power-up timer: toon een kleine aftelbalk onder de speler wanneer een power-up actief is
3. Stijgende moeilijkheidsgraad: voeg bij level 5+ een tweede obstakel tegelijk toe per spawn
4. Score-multiplier indicator: toon een kleine x2/x3 badge naast de score wanneer een combo actief is
