# Game Taken

1. Voeg vallende obstakels toe die de speler moet ontwijken
2. Voeg een score-teller toe die elke seconde oploopt
3. Voeg een game-over scherm toe wanneer de speler een obstakel raakt
4. Voeg een startscherm toe met een "Start" knop
5. Voeg geluidseffecten toe (beep bij botsing) via Web Audio API
