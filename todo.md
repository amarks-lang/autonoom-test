# Game Taken

1. Voeg een countdown-timer toe (bijv. 30 seconden) als extra speelmodus: "Tijdaanval" waarbij de speler zoveel mogelijk punten haalt voor de tijd op is
2. Voeg een level-select scherm toe op het startscherm waarmee de speler direct op een hoger level kan beginnen (ontgrendeld na behalen van dat level)
3. Voeg een magnetisch effect toe: als de speler dicht bij een muntje komt (binnen 40px), wordt het naar de speler toe getrokken
4. Voeg een scorebord-animatie toe op het game-over scherm: de top-3 scores verschijnen één voor één met een teller-effect omhoog
5. Voeg een "dubbel springen" mechanic toe: de speler kan tweemaal omhoog bewegen voor een hogere boost, met een kleine cooldown indicator
