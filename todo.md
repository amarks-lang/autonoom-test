# Game Taken

1. Voeg een "combo multiplier" toe: elke 5 punten zonder geraakt te worden verhoogt de multiplier (x1 → x2 → x3), toon het op het scherm
2. Voeg een korte schudanimatie toe aan het canvas als de speler geraakt wordt (screen shake)
3. Toon een mini-kaart of radarpuntje van inkomende obstakels rechtsboven in de hoek
4. Voeg een visuele "trail" effect toe bij speedboost: kleurrijke deeltjes achter de speler
5. Voeg een "perfecte ontwijking" melding toe als de speler een obstakel binnen 2px passeert (beter dan near miss)
